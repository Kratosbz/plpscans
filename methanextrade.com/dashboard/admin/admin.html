API Endpoint Reference
POST /auth/login    — User login → returns JWT token
POST /auth/register  — New user signup
POST /auth/admin/login — Admin login → returns JWT

POST /users/:id/profit/add  — Add profit to user
POST /users/:id/profit/deduct — Deduct profit
GET  /users/:id/profit     — Get profit + history

GET  /deposits         — All deposits (admin)
POST /deposits        — Submit deposit (user)
PUT  /deposits/:id/status — Approve / decline

GET  /withdrawals      — All withdrawals (admin)
POST /withdrawals     — Submit withdrawal
PUT  /withdrawals/:id/status — Approve / decline

GET  /traders         — All traders
POST /traders        — Create trader
PUT  /traders/:id     — Update trader
DELETE /traders/:id    — Delete trader
POST /traders/:id/connect — Connect user to trader
POST /traders/:id/history/:userId — Add trade to user history
GET  /traders/:id/history/:userId — Get user's trade history
